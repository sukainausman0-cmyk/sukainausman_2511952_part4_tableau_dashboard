# Chart Selection Justification

## Sales Trend View
Question: How are sales and profit changing over time? A line or time table is appropriate because months have a natural order. Color separates sales from profit. The design avoids pie charts because time trends need continuity.

## Regional Performance View
Question: Which regions perform best? A sorted bar/highlight view is appropriate because leaders need fast comparison across regions. Color and sorting emphasize strongest and weakest regions. The design avoids unsorted regional labels.

## Category Profitability View
Question: Which categories and sub-categories drive profit or risk? A bar/highlight table is appropriate because it compares sales, profit, returns, and margin together. The design avoids decorative charts that hide negative profit.

## Customer Segment View
Question: Which segment contributes most? A bar comparison is appropriate because customer segments are discrete categories.

## Shipping Performance View
Question: Which shipping modes create delay risk? A mode and delay bucket view is appropriate because it shows operational causes clearly.

## Discount vs Profit View
Question: How does discounting affect profit? A scatter or bucket comparison is appropriate because discount is a continuous driver and profit is a financial outcome. The dashboard avoids implying causation from correlation alone.

## Return Analysis View
Question: Where are returns concentrated? A category/region/segment comparison is appropriate because returns need operational follow-up.
